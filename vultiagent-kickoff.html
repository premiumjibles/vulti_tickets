Phase 3 / Weeks 9–12

Autonomous Agents

The wallet evolves from a tool you operate into a system that acts on your behalf, within user-defined guardrails.

  • Agents as TSS signers, M-of-N signing landed in Phase 2. Now agents hold key shares and participate in signing sessions just like devices
  • Agent runtime, Containerized AI agents as persistent processes. One-click hosted deploy or self-hosted via Docker Compose
  • Configurable guardrails, Custom system prompts with enforced limits: max tx size, asset whitelists, daily spend caps, cool-down periods
  • Adversarial co-signing, Two agents with different mandates must both approve. Growth vs. risk agent debate protocol
  • Logged reasoning, Full audit trail of every debate, argument, and decision for user review and trust-building

Delivery Rhythm

Lightweight coordination designed for speed. Automated visibility so engineers stay heads-down shipping.

  • 2-week mission cycles, work is shaped into goal-oriented missions with clear scope and timeboxes, assigned at the start of each cycle
  • Daily automated pulse, a bot summarizes each engineer's git activity (PRs, branches, areas touched) and posts to Discord. Zero manual reporting
  • Weekly 30-min sync, course corrections, architectural decisions, and blockers only. Not a status update
  • Self-test, self-merge, CI enforces quality (lint, types, tests under 2 minutes). Engineers ship without waiting on reviews. Critical areas like signing and key material require pre-merge review
  • Lightweight mission tracking, GitHub Issues for mission-level visibility and milestones. No heavyweight ticket management

Cost & Pricing

How the Routing Works

Three models, one routing layer. The model upgrades with the operation complexity.

Gemini Flash, $0.003/op

Chat, balances, sends, simple swaps, token lookups. ~70% of all traffic. Free & unlimited for everyone.

Claude Haiku, $0.03/op

Cross-chain swaps, staking, optimal routing. Premium ops on Free & Pro tiers. 100% success, 2 avg loops.

Claude Sonnet, $0.13/op

Stop-loss, TWAP, dust sweeping, autonomous agents. Premium ops on Power tier. Best tool-calling accuracy.

Cut Prompt Size 50%

82% of every request is fixed overhead. Trim system prompt + filter tools by intent. Halves all costs. Ships Phase 1.

Anthropic Prompt Caching

Direct API migration. System prompt prefix identical across all users,65% further reduction on Haiku/Sonnet. Ships Phase 2.

Swap Affiliate Revenue

0.3% on THORChain/DEX swaps → ~$1.50/active user/month. Covers 2× free tier cost. Revenue scales with usage.

LLM costs have dropped ~10× in 18 months. Architecture is model-agnostic, provider swaps are a config change.

Uncapped Curve

Formula: (staked / 12)0.75 × lock multiplier. Never caps — every additional VULT earns more. K=12 is a governance parameter, adjustable over time.

Time-Lock Multiplier

30-day = 1×, 90-day = 1.3×, 180-day = 1.6×. Rewards conviction, discourages mercenary capital. Modeled after Curve/1inch ve-tokenomics.

Token Flywheel

Buy pressure + supply removal from staking. 25K VULT staked 180 days = 500 bonus ops ($50/mo value) → ~19% annual yield at current price. Funded by real revenue.

Stacking example: Pro ($10/mo, 100 ops) + 100K VULT staked 180 days = 1,511 premium ops. Subscriptions + staking complement, never compete. Tier features (stop-loss, TWAP, agents) still require upgrade.

Cost & Pricing

VULT Token Economics

Staking bonus valued via dividend discount model. Premium op = $0.10 (Pro per-op rate). DIEM ($1/day inference per token) as market reference.

Yield at Current Price

25K VULT staked 180 days → 500 ops/mo → $600/yr on $3,175 locked = ~19% annual yield. DIEM equilibrium ceiling: ~18%. Right in the zone.

Dividend Discount Model

Fair Value = Annual Ops Value / Required Return. At 20% required return: $0.12/VULT theoretical floor from staking utility alone. Current price: $0.127.

In-Kind Discount

Markets value in-kind benefits at ~40% of cash (DIEM: $800 price vs $2,000 theoretical). Staking utility provides a price floor, not a ceiling. Speculation + adoption add on top.

Reflexive Loop

Generous yield → users buy VULT to stake → circulating supply drops → price rises → yield compresses (%) but absolute value holds → supply reduction amplifies further. This is the DIEM playbook: $1/day generosity triggered a reflexive loop from single digits to $800.

Cost Sustainability

Worst-case whale (1M VULT, 180-day): ~$240/mo LLM cost for $127K locked = 2.3%/yr. In practice, 40–60% op utilization halves real cost. Funded by subscription + affiliate revenue, not token inflation. K is adjustable via governance.

Strategy: launch generous (K=12) to bootstrap staking demand and supply shock. Adjust K after 6 months of utilization data. Start aggressive, tighten with data.

Builder-Led Marketing

Engineers Are the Brand

In crypto, technical depth signals legitimacy, polished marketing signals scam. Monad, Eigenlayer, and Uniswap all grew through engineers posting technical content. Our version for a 4-person team:

@vultisig: 1 Thread / Week

Every Friday, one "what we shipped this week" thread with screenshots or a demo clip. Owned by the community manager (see below), sourced from engineers. This single artifact does more than daily filler posts.

Engineers: Post When It's Interesting

No schedule, no quota. When you hit a cool bug, a nice before/after, or a wild edge case, post it. Tag @vultisig. Some weeks that's 3 posts, some weeks it's zero. Authenticity > consistency.

"Building @vultisig" in Bios

Vulti Agent is a sub-product of Vultisig, one brand, one audience. Every engineer's bio links to @vultisig, and a pinned tweet explains what Vulti Agent is. Every personal post passively drives discovery back to the product.

Quote-Tweet > Retweet

When @vultisig amplifies engineer posts, quote-tweet with added context ("our MPC engineer just shipped X"). Quote tweets get 2–3× the reach of plain retweets. This is the entire amplification playbook.

Who Runs @vultisig Day-to-Day

A dedicated community manager / marketing-ops hire. Writes the weekly ship thread, handles replies and DMs, quote-tweets engineer posts, manages Discord community. Engineers provide the raw material (screenshots, clips, technical context); the CM shapes and publishes it. This is a standalone role, not an engineer's side job. Budget: ~$3–5K/mo for a crypto-native CM (part-time or contractor). Can start Phase 1.

Quality over quantity. One substantive thread per week beats seven filler posts, crypto audiences are saturated with hype and actively filter it out.
